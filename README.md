# 🌱 Crop Data Visualization using Python

## Project Overview
This project focuses on analyzing crop data using Python and visualizing key agricultural patterns.

## Dataset
- Source: Kaggle
- Observations: 2200
- Features: N, P, K, Temperature, Humidity, pH, Rainfall, Crop Label

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Histograms
- Line Charts
- Bar Charts
- Pie Charts
- Scatter Plots

## Key Insights
- Optimal pH range: 6–7
- No strong relation between rainfall & temperature
- Strong clustering in nutrient values (P & K)

##  Conclusion
Dataset supports crop recommendation based on environmental and soil conditions.
